下面是每个步骤的一些详细说明和注意事项：

🛠️ 从零开始游玩ATM10整合包指南

1. 准备工作和软件下载

在开始之前，你需要先下载一些必要的软件和文件。请注意，ATM10整合包通常需要Java 17或更高版本的支持。

  • Java 17: 你可以从 Adoptium (Eclipse Temurin) 或 Azul Zulu 等网站下载。下载后运行安装程序，通常只需一路点击“下一步”即可完成安装。
  • PCL2启动器: 这是一款在国内网络环境下下载速度较快、对整合包支持良好的启动器。你可以从它的官方爱发电页面或其他可靠来源获取。
  • ATM10整合包: 整合包通常可以在 CurseForge 或 Modrinth 等网站上找到。在这些网站的搜索框中输入 “All the Mods 10” 或 “ATM10”，找到项目页面后，选择下载 客户端版本（通常标注为 Client 或 Windows）。

2. 安装与配置PCL2启动器

  1. 安装PCL2: 将下载好的PCL2启动器压缩包解压到一个你方便找到的文件夹中（不要解压在系统盘或桌面），然后运行启动器。
  2. 登录账户: 启动PCL2后，你需要登录你的Minecraft账户。
    • 如果你拥有正版账户，请选择“微软”登录方式，并按照指引完成验证。
    • 如果没有正版账户，可以选择“离线”模式，然后设置一个任意用户名（不含特殊字符）即可游玩。
  3. 安装ATM10整合包: 在PCL2启动器的界面上，找到“下载”或类似选项。然后，直接将你下载好的ATM10整合包文件（.zip格式）拖拽到PCL2的窗口中。启动器会自动识别并开始安装。这个过程会下载所需的Minecraft版本、Forge/Fabric（ATM10很可能基于Forge）以及所有模组文件，请保持网络通畅并耐心等待。

3. 获取并安装柠娜汉化包

ATM10整合包本身是英文的，你可以使用柠娜制作的汉化包。

  1. 获取汉化包: 在哔哩哔哩（Bilibili）上搜索UP主“柠娜”（UID: 383989569）。在她的空间里，通常会有视频或动态提供一个包含了数百个热门和冷门整合包汉化的文檔链接（例如一个在线表格）。在这个文檔中按 Ctrl + F 搜索“ATM10”或“All the Mods 10”，找到对应的汉化资源包下载链接。
  2. 安装汉化包:
    • 下载汉化包，它通常是一个 .zip 格式的资源包文件，不需要解压。
    • 启动Minecraft游戏，进入主菜单后，点击选项（Options） -> 资源包（Resource Packs）。
    • 点击“打开资源包文件夹（Open Pack Folder）”，这会打开游戏存放资源包的文件夹。
    • 将你下载的汉化包 .zip 文件复制或移动到此文件夹中。
    • 返回游戏，汉化包应该会出现在“可用资源包”列表中。点击箭头将其添加到“已选资源包”栏，并拖动到列表顶端以确保其优先加载。
    • 点击“完成（Done）”，游戏会重新加载资源，界面和模组内容应该就已变成中文了。

4. 启动游戏与优化

  1. 分配足够内存: ATM10这类大型整合包需要较多的运行内存（RAM）。在PCL2启动器的版本设置或游戏设置中，找到内存分配（可能叫“Java设置”、“最大内存”等）。建议至少分配6GB（6144MB）或以上的内存，分配8GB（8192MB）会更流畅。但注意不要超过你电脑物理内存总量的80%。
  2. 启动游戏: 返回PCL2主界面，在版本选择下拉菜单中确保选择了“ATM10”，然后点击“启动游戏”。
  3. 调整视频设置（可选）: 首次进入游戏，由于模组众多，可能会比较卡顿。你可以根据电脑性能，在游戏的视频设置（Video Settings） 中适当调低一些选项，例如：
    • 画质（Graphics）: 从“精美（Fancy）”调为“流畅（Fast）”
    • 模拟距离（Simulation Distance） 和 渲染距离（Render Distance）: 适当调低（如12 chunks以下）
    • 光影（Shaders）: 初期可不加载或选择性能要求低的光影
    • 关闭垂直同步（VSync） 和 Mipmap级别
      调整完毕后记得点击“**完成（Done）”**或“应用（Apply）”。

完成以上所有步骤后，你应该就可以顺利开始你的ATM10之旅了！如果遇到任何问题，比如游戏崩溃，可以尝试查看游戏日志文件（logs文件夹）或崩溃报告（crash-reports文件夹）来寻找线索。

祝你玩得开心！
