3. 博客用户表添加标识列

语法

alter table list_name add column_name datatype identity(1,1)

column_name datatype identity(1,1) null/not null)

注意

GUI中 表设计器在修改标识列属性时不允许直接更改

  • 解决方法：工具->选项->设计器->表设计器和数据库设计器

五、数据查询

1.博客系统基础查询

语法

select [all|distinct]字段列表
[into]新表 from数据源 [where]条件表达式 
[group by]分组表达式 [having]搜索表达式 
[order by]排序表达式 [asc|desc]
`select * from list_name`查询表中所有信息
`select column1,column2 from list_name`查询指定信息
`select column1 as Name1,column2 as Name2 from list_name`查询指定信息并按各列取列名
`select xxx(同上) where column = 'name'`查询对应元素的信息
`select * from list_name where column is null`查询对应元素为空的信息
`select top 3 * from list_name`查询前三位

Name1无需添加“” 可以加，可读性这一块
null查找不能“=”,使用“is null”或“is not null”

六、高级查询的应用

  • 笛卡尔积
  • sql连接查询
from join_table join_type join_table
[on (join_condition)]

join_table进行连接操作的表名
join_type连接类型
on(join_condition)连接条件

  • 子查询

1.查询博客的完整信息

实例

select B.blogNO as 编号, B.blogTitle as 标题 
from blogUser as U inner join blog as B 
on U.UserNo = B.uerNO

2.查询博客的完整信息并把未发表博客的用户信息显示出来

即使用外连接，left join左外连接，right join右外连接，full join全外连接

实例

select B.blogNO as 编号, B.blogTitle as 标题 
from blogUser as U (left/right) join blog as B 
on U.UserNo = B.uerNO

3.简单子查询——按博主查询博客

实例

#法1：连接查询
select B.* from blog as B inner join blogUser as U
on U.UserNo = B.uerNO and U.userName = '名1'

#法2：子查询
select * from blog 
where uerNO = (select UserNo from blogUser where userName = '名1')

子查询释义：where里查询bloguser中username为‘名1’的userNO，再查询blog里uerNO = userNO的所有数据

4.in子查询

实例

select userName from blogUser where UserNo 
in (select distinct uerNO from blog)

1.在blog里查询了userno
2.在bloguser里查询userNO第一步对应的username

disstinct去掉重复元素
not in即‘不在第一步’

5.exists查询——标记冷门博客

实例

update blog set blogTitle = '(冷门博客)' + blogTitle
where exists (select * from blog where clickNum = 0)
and clickNum < 10

释义：exists如果存在clickNum=0,则将clickNum<10的blogTitle改名

declare用于声明局部变量

declare @name type
声明多个只需添加逗号

使用set(或select)赋值

set @name = value

区别项 set select
同时赋值多个 否 是
表达式返回多个值 出错 收到最后一个值
表达式未返回值 赋值为NULL 保持原值

1.变量

实例

declare @username varchar(20)
--声明一个局部变量
set @username = '名2'
--将变量赋值为 名2
select * from blogUser where userName = @username
--查询符合blogUser里userName符合的全部信息

2.if条件

语法

#begin语句
begin
	{sql_statement|statement_block}
end

#if语句
if boolean_expression
	{sql_statement1|statement_block1}
else
	{sql_statement2|statement_block2}

实例

declare @number int
set @number = 50
if @number > 100
	print 'xxxxx>100'
else
	begin
		if @number < 10
			print 'xxx<10'
		else
			print 'xxx>10<100'
	end

3.while循环

语法

while boolean_expression
	{sql_statement|statement_block|break|continue}

实例

declare @num int = 1
while @num <100
	begin
		print @num*2
		set @num = @num +1
	end

4. case多分支

实例

declare @level char(1)
set @level = 'C'

select '成绩' = 
	case @level
		when 'A' then '成绩90+'
		when 'B' then '成绩80+'
		when 'C' then '成绩70+'
		else 'NO case'
end
